NextGen Gallery，影集

今天我给 FunnyWork 新增了一个 Albums 页面，具体效果大家可以自行查看。这个相册是利用 Wordpress 插件 NextGen Gallery 实现的，那么现在就来教教大家怎么来新建一个新的影集。

首先登录到 Wordpress 后台，点击网站名称下面的“影集”进行管理。一进来便是 NextGen Gallery 的概览页面，橙色的工具条右端有很醒目的两个按钮：

image

顾名思义，前面一个按钮是管理影集，而后面那个就是上传图片了，点击它进入到下一步：添加一个新影集。

image

新影集的名称不允许出现汉字，不过无需担心，这个只是用来命名图片所在的新文件夹，影集显示的标题是可以在以后任意修改的。

建好影集之后，就可以向影集里面添加图片了，添加图片有三种途径：

  1. 上传一个ZIP文件，NextGen 会自己解压缩这个包里所有的图片。不过需要特别提醒的是，Winrar 默认的压缩格式.rar 是不支持的哦；
  2. 导入图片文件夹。用 FTP 的同学有福了（这个……貌似只有俺自己有 FTP 权限-_-），你可以直接使用 FTP 把图片上传到某个文件夹，而且不一定就得传到 NextGen 默认的 /wp-content/gallery/ 下面，这个文件夹是可以任意指定的。你甚至可以指定 Wordpress 默认的上传文件夹，比如 /wp-content/uploads/2008/04/，这样就能显示2008年4月上传的所有照片了，很强大吧？不过千万要慎重，某次我在勾选了“当删除影集时删除文件”（该选项可以在 NextGen 的选项里面修改）的情况下删除了这么一个影集，直接导致日志里面所有图片都不能显示的悲惨后果……
  3. 最后一个，也是最土鳖的方法：上传图片。这个就不用多介绍了，记得选择相应的相册即可。

不要以为上传好了照片就万事大吉，这个还是有点小小的麻烦的，你需要转到“管理影集”的页面，对你刚建立好的影集进行操作，否则它是不会显示在 Albums 页面上的。

进入某个影集的修改界面，默认是该影集下面所有的图片列表，你可以修改图片的标题、描述，或者删除它。还可以在选中图片ID之后对其进行如下操作：

image

如果你在 NextGen Gallery 的“选项”页面激活了相关图片功能的话，那有一个功能你必须了解一下，就是“显示标签”：

image

给图片定义标签之后，比如给某张图片定义了一个标签“女王”，那么在有“女王”标签的日志页面上就会显示这张图片了。一张图片可以使用英文逗号“,”同时定义多个标签。

以上是影集内图片的编辑选项，那么整个影集该怎么编辑呢？这个比较隐蔽，在影集标题的后面有这么一个按钮：

image

就是它了。点击会展开影集的相关选项，一共是六个：

  1. 标题。
  2. 描述。
  3. 路径。
  4. 页面链接到。
  5. 预览图像。
  6. 创建新页面。

刚创建好的影集你需要为它创建一个新页面（第6项），记得将新页面的父页面设置为“Albums”。

image

然后再设置它的链接页面（第4项），就是你刚才创建的那个新页面，否则在 Albums 页面点击影集封面会没有反应。

好了，上述几项都填写完毕，保存更改。是不是觉得大功告成了？到 Albums 瞅瞅，倒，为什么还是看不到我创建的相册呢？哈哈，还有最后一步，点击 NextGen Gallery 菜单中的“相册”，进入管理相册的界面，是不是发现你的影集出现在右侧的列表框中呢？

一般情况下，你不需要新建一个相册，直接选择"default"相册即可：

image

然后把你的影集直接拖拽到左侧的“default”相册框内，点击右上方的“更新”按钮，这回就真的OK了！

最后补充一下在日志上调用 NextGen Gallery 的方法：

  • 插入一个幻灯片：[slideshow=gallery_id,width,height]
  • 插入一个相册：[album=album_id,extend] 或者 [album=album_id,compact]
  • 插入一个影集：**gallery=_gallery_id
    _**请自行添加“[ ]”
  • 插入一张图片：[singlepic=pic_id,width,height,mode,float]
    * mode 是可选项，可选参数有 web20（倒影效果），watermark（水印效果）
    * float 也是可选项，可选参数有 left（左对齐），right（右对齐）
  • 插入一个图片浏览器：**[imagebrowser=gallery_id]
    **图片浏览器的效果如下所示：
    image

如果你有给照片添加标签的好习惯，那么还有下面两种方法：

  • 插入和一个或者几个标签关联的所有图片：tags=tagname1,tagname2,tagname3…
  • 以相册的形式组织和这些标签关联的图片：**albumtags=tagname1,tagname2,tagname3…
    *** 请自行添加“[ ]”

更详细的说明请参考 NextGen Gallery 的 FAQ 页面：http://wordpress.org/extend/plugins/nextgen-gallery/faq/
